# Dragon Assault

Dragon Assault is a simple side-scrolling shooter game. The game was built using [Phaser 3](https://phaser.io/).

![Dragon Assault Game Screenshot](screenshot.png)

## Installation

Clone the repo, then run the below commands.

```bash
npm install
```

```bash
npm run dev
```
